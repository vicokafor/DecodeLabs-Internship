
# Project 2: Exploratory Data Analysis (EDA)

## Overview
This project was completed as part of the DecodeLabs Data Analytics Internship Program. The objective was to explore and analyze a cleaned e-commerce dataset to uncover patterns, trends, outliers, and business insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Descriptive Statistics
- Outlier Detection using Box Plots
- Revenue Distribution Analysis
- Product Performance Analysis
- Payment Method Analysis
- Referral Source Analysis
- Order Status Analysis
- Monthly Revenue Trend Analysis
- Correlation Analysis

## Key Findings
- Most orders were concentrated in lower price ranges, resulting in a right-skewed distribution.
- High-value transactions were identified as legitimate business transactions.
- Chairs generated the highest sales volume and revenue.
- Credit Card transactions generated the highest revenue.
- Instagram was the strongest referral channel in both order volume and revenue.
- Revenue peaked in June before declining in the second half of the year.
- UnitPrice and Quantity were the strongest drivers of TotalPrice.

## Recommendations
- Increase investment in Instagram marketing campaigns.
- Investigate causes of order cancellations.
- Prioritize high-performing products.
- Implement product bundling strategies.
- Encourage credit card payments through incentives.
